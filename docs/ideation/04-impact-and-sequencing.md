## Dependency notes

- **Foundation chain:** FIX-04 (cache dedupe/TTL/migrations) → FIX-02
  (incremental ingest) → FIX-03 (entity resolution) → FIX-01 (live path).
  FIX-01 can start in parallel but should not merge before FIX-04/FIX-07.
- **Guard-before-growth:** FIX-07 (egress guard) must precede or accompany
  anything that adds network code (FIX-01, EXP-07, EXP-08, EXP-12).
- **Measure-before-show:** FIX-05 (exposure metrics) precedes EXP-01 (panel),
  EXP-10 (writeup), and strengthens FIX-06 (eval realism).
- **Model-before-extract:** EXP-03 (LensSpec) precedes EXP-09
  (shared library); EXP-09 also needs a queer-the-stacks code comparison that
  this pass did not perform.
- **FIX-10 (conflict/corrections) ⇄ FIX-14 (confidence tiers) ⇄ EXP-05
  (fix-at-source)** form one coherent "identity honesty" bundle — cheapest
  done together.
- **Scale-sensitive:** FIX-13 (perf) and EXP-04/EXP-06 are only meaningful
  after FIX-02 produces real-sized profiles.
